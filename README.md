# Basic
Homemade project for creating web app for collecting data about stamps

# Installation
## Init DB
```
python backend/init_db.py
```

## Run BackEnd
```
python -m uvicorn backend.main:app --port 8080 --reload
```

## Run FrontEnd
```
 $ cd frontend
 $ npm run serve
```
